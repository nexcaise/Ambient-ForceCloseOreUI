# ForceCloseOreUI

A handy utility to force close the Minecraft OreUI.

---

## How to Use

### 📱 Android

#### 1. Download the Latest SO File

- Go to the [Releases page](https://github.com/NexCaise/Ambient-ForceCloseOreUI/releases) to download the latest `.so` file.

#### 2. Installation

- **If you are using [Ambient](https://ambient.kitsuri.dev):**
  1. Download the `.so` file.
  2. Tap the file and choose to open with Ambient for import.
  3. Configuration file location:
     ```
     /storage/emulated/0/Android/media/[ambient package name]/modules_config/ForceCloseOreUI
     ```
     > **Note:** Modified APK files are NOT provided.
